# MusicBox

1. install & build

```sh
yarn
yarn build
```

2. (optional) custom config

- create `config.json`
- fill [typeorm config](https://github.com/typeorm/typeorm/blob/master/docs/connection-options.md)

3. run

```sh
node dist
```

4. usage

open http://127.0.0.1:12356
use `admin` / `admin` to login
